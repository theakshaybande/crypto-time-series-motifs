# Source Code

This folder contains the thesis codebase.
- `data_collection/`: scripts to download and ingest market data.
- `data_processing/`: preprocessing, cleaning, and transformation code.
- `motif_methods/`: implementations/wrappers of motif discovery methods.
- `regime_detection/`: market regime identification modules.
- `benchmarking/`: experiment orchestration and metric computation.
- `visualization/`: plotting and reporting utilities.
